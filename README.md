# 🎵 Music App

玻璃态 UI 音乐应用，包含搜索、播放、我的三个页面！

## 📱 没有电脑？如何获取 APK！

### 方案 1：使用 Expo 在线构建（最简单！）

1. **注册 Expo 账户**（手机浏览器访问 https://expo.dev/signup）
2. **克隆仓库** 到你的 GitHub
3. **使用 EAS Build 在线构建**：
   - 访问 https://expo.dev
   - 登录后创建项目
   - 导入这个 GitHub 仓库
   - 点击 "Build" → "Android" → "Create build"
4. **等待构建完成**（约 10-20 分钟）
5. **下载 APK**！

### 方案 2：找朋友帮忙（如果有 Android Studio 的朋友）

把这个仓库链接发给朋友，让他们用 Android Studio 打开并构建。

### 方案 3：使用在线 Android 构建服务

找在线构建服务（如 Appcircle），导入 GitHub 仓库直接构建！
